üî¢ Essential Formulas

Lookup

VLOOKUP

Find and retrieve data from a table:

=VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])
Example: =VLOOKUP(A2, DataTable, 3, FALSE)
Pro tip: Always use FALSE (or 0) for exact match
Advanced Lookup

INDEX + MATCH

More flexible and faster than VLOOKUP - can look left!

=INDEX(return_range, MATCH(lookup_value, lookup_range, 0))
Example: =INDEX(B:B, MATCH(A2, A:A, 0))
Logic

IF Statement

Handle multiple conditions with nested IF:

=IF(condition, value_if_true, value_if_false)
Nested Example: =IF(A1>90, "A", IF(A1>80, "B", IF(A1>70, "C", "F")))
Conditional Math

SUMIF & COUNTIF

Sum or count based on specific criteria:

=SUMIF(range, criteria, [sum_range])
=COUNTIF(range, criteria)
Example: =SUMIF(A:A, ">100", B:B)
Text

Combine Text

Join text from multiple cells:

=A1&" "&B1
=CONCATENATE(A1, " ", B1)
Modern way: Use TEXTJOIN() for multiple cells with delimiter
Error Handling

IFERROR

Handle errors gracefully in your formulas:

=IFERROR(formula, value_if_error)
Example: =IFERROR(A1/B1, "Division error")
