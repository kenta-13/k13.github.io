Navigation

Quick Navigation

Navigate File Explorer like a pro:

Ctrl + N (New window)

Quick Access

Open Terminal Here

Open Command Prompt or PowerShell in current folder:

  • Type "cmd" in address bar and press Enter (Command Prompt)
  • Type "powershell" in address bar and press Enter (PowerShell)
  • Right-click folder ‚Üí Show more options ‚Üí Open in Terminal (Windows 11)
Bonus: Type "wt" for Windows Terminal
File Management

Batch Rename Files

Rename multiple files at once:

  1. Select multiple files (Ctrl + Click)
  2. Press F2
  3. Type new name
  4. Files will be named: Name (1), Name (2), etc.
Pro tip: Use PowerShell for advanced renaming with patterns

  • View ‚Üí Show ‚Üí File name extensions
  • Or: Alt ‚Üí View tab ‚Üí Options ‚Üí View ‚Üí Check/uncheck settings
Why: Essential for development and security awareness

Maintenance

Free Up Disk Space

Clean up unnecessary files automatically:

  1. Settings ‚Üí System ‚Üí Storage
  2. Enable Storage Sense for automatic cleanup
  3. Click "Temporary files" to manually delete
  4. Or: Win+R ‚Üí cleanmgr ‚Üí Select drive
Pro tip: Click "Clean up system files" for additional options
System

Advanced Task Manager

Monitor and manage system performance:

Ctrl + Shift + Esc (Open Task Manager)
  • Performance tab: Real-time CPU, RAM, disk usage
  • Startup tab: Manage startup programs
  • Right-click process ‚Üí End task to close frozen apps
Package Manager

Windows Package Manager (winget)

Install apps via command line:

winget search app_name (Search for app)
winget install app_name (Install app)
winget upgrade --all (Update all apps)
Example: winget install Google.Chrome
